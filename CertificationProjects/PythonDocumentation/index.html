Introduction

Python is a high-level, interpreted programming language that emphasizes code readability and simplicity. Created by Guido van Rossum and first released in 1991, it has become one of the most popular languages in the world due to its versatility and ease of use.

Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming. Some of its key features include:

  • Simple and easy-to-learn syntax
  • Extensive standard library
  • Dynamically typed variables
  • Cross-platform compatibility
  • Open-source community support
# Hello World in Python
print("Hello, World!")
Variables and Data Types

In Python, variables are dynamically typed, meaning you don’t need to declare their type before assigning a value. Some of the core data types used in Python are:

  • Integers (int)
  • Floats (float)
  • Strings (str)
  • Booleans (bool)
# Example of variables and data types
x = 5 # integer
y = 3.14 # float
name = "Python" # string
is_cool = True # boolean

Python also supports complex data structures like lists, tuples, dictionaries, and sets, which help organize and manipulate data efficiently. These data types are flexible and can hold different types of values.

Control Flow

Python offers standard control flow structures such as if-else, for loops, and while loops, which allow you to make decisions and repeat operations based on conditions. Control flow is essential when automating tasks and handling decision-making processes in programs.

In addition to conditional statements and loops, Python offers logical operators like:

  • not
  • or
  • and

These operators are used to combine or negate conditions.

x = 10
if x > 5:
print("x is greater than 5")
else:
print("x is 5 or less")
Functions

Functions in Python are defined using the def keyword and are used to encapsulate reusable code. They can take parameters and return values, making them a powerful tool for writing modular code. This modularity enables better organization and reuse of code.

# Example of a Python function
def greet(name):
return f"Hello, {name}!"

print(greet("Alice"))

Python also supports anonymous functions, known as lambdas, which are small, single-line functions. Lambdas are useful for short operations that don't need a full function definition.

Lists and Loops

Lists are a fundamental data structure in Python, allowing you to store and manipulate ordered collections of items. Python makes it easy to iterate over lists using loops, which simplifies working with sequences of data.

Common list operations include adding or removing items, sorting, slicing, and filtering. Python’s list comprehensions provide a concise way to create lists based on existing ones, making loops more efficient and readable.

# Example of list and loop in Python
numbers = [1, 2, 3, 4, 5]
squared = [n**2 for n in numbers]
print(squared) # Output: [1, 4, 9, 16, 25]
